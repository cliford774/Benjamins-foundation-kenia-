# Benjamin's Foundation Kenia

Welcome to the official website for Benjamin’s Foundation Kenia.

Our mission is to support vulnerable communities through education, volunteer opportunities, women empowerment, and sustainable community development.

## Features
- Volunteer information
- Donation support
- Community impact stories
- Project updates
- Contact and partnership opportunities

## Technology
- HTML
- CSS
- JavaScript

## Purpose
To create a transparent and accessible platform that connects supporters, volunteers, and communities.

Building pathways to opportunity.
